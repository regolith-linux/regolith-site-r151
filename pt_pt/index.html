O Regolith é um ambiente de desktop moderno projetado para permitir que você trabalhe mais rapidamente, reduzindo a confusão e a cerimónia desnecessárias. Construído em cima do Ubuntu, GNOME, e i3, o Regolith assenta numa base bem suportada e consistente.

Obter o Regolith Saber mais Contribua

Visita Visual

Após o primeiro login, uma sobreposição (mostre a qualquer momento com super ?) apresenta os atalhos de teclado mais importantes usados com i3-wm.

Para aqueles que trabalham com o terminal, pressionando super enter é tudo o que é preciso para chegar a ele.

Um lançador global e único de aplicações está disponível instantaneamente e em qualquer lugar para executar os seus programas via super space.

Necessita de mais terminais? Crie layouts rapidamente alternando entre os modos horizontal e vertical com super backspace seguido de super enter. Navegue para janelas posicionalmente com super h j k l.

Mude para o modo de janela flutuante com super F. Redimensione janelas com super r e mova-as com o rato pressionando super.

O Gnome Flashback fornece uma gestão do sistema consistente e simples. Ajuste o seu interface, monte automaticamente as suas unidades USB, ligue-se a redes wireless. Inicie o painel de controlo com super c.

Mude facilmente para Solarized ou outros temas com o comando regolith-look. Devido ao looks utilizar o gestor de pacotes, você armazena apenas recursos do tema que está a utilizar.

O Regolith é construído em cima do Ubuntu, fornecendo um sistema atualizado e estável.

Muito a acontecer? Encontre rapidamente a janela que procura através de super ctrl space ou navegue através entre espaços de trabalho com super [0 - 19].

Não desperdice espaço de interface e aproveite cada pixel sem ter que gerir minuciosamente os seus layouts de janelas.

As notificações do ambiente de trabalho não competem pela sua atenção, sendo geridas através de uma caixa de diálogo pressionando super n.

O que torna o Regolith diferente

  • Oferece um desktop com um interface de utilizador funcional, mas mínimo, que pode ser customizado e expandido se necessário.
  • Combina os recursos de gestão de sistema do GNOME com o fluxo de trabalho produtivo do i3-wm.
  • Permite aos novos utilizadores uma maneira rápida e divertida de experimentar um gestor de janelas lado a lado.
  • Suporta customizações fáceis e ricing através de uma configuração Xresource consistente.
  • Conta com a loja de aplicações e repositórios de pacotes do Ubuntu para oferecer uma grande variedade de software de alta qualidade.
  • Construído para ser modificado. Troque componentes de interface a seu gosto facilmente.
  • É fornecido com uma janela sobreposta de atalhos para facilitar a iniciação.
  • Fornece um script de construção e metadados de pacotes para permitir que os utilizadores explorem facilmente o ambiente de trabalho e a distribuição.
